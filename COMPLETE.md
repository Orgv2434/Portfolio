## 💡 快速自定义指南

### 修改个人信息
在 `portfolio.html` 中找到：
```javascript
<h1>Portfolio</h1>           // 改为你的名字
<p>游戏专业 全栈创意人</p>   // 改为你的职位
```

### 添加新项目
```javascript
{
    title: '项目名',
    description: '项目描述',
    tags: ['标签1', '标签2'],
    image: { emoji: '🎮', color1: '#ff6b6b', color2: '#ee5a6f' }
}
```

### 修改背景颜色
```css
/* 找到这行并修改颜色 */
background: linear-gradient(-45deg, #新颜色1, #新颜色2, #新颜色3, #新颜色4);
```

### 调整动画速度
```css
/* 改这个时间值（单位：秒） */
animation: aurora 15s ease infinite;  /* 15s 改为你想要的 */
```

## 🔧 常见问题速答

**Q: 需要安装什么？**  
A: 如果只是查看效果，不需要安装任何东西，直接打开 portfolio.html

**Q: 颜色太亮了？**  
A: 编辑 portfolio.html 中的颜色值，改成你喜欢的

**Q: 如何添加项目链接？**  
A: 在卡片点击时添加 `onClick={() => window.open('链接')}`

**Q: 移动端显示不正常？**  
A: 强制刷新浏览器（Ctrl+Shift+Delete）清除缓存

**Q: 想要更多动画？**  
A: 使用 Framer Motion 库，或查看 styles.css 添加更多 keyframes
